
Три двойные кавычки (""") в Java используются для создания блоков текста
Основное использование:
1. Избежать экранирования (Escaping) специальных символов, таких как двойные кавычки (") и символы новой строки (\n)
2. Сохранить форматирование текста, включая пробелы и переносы строк, как они записаны в коде
Пример:
До тройных кавычек
String jsonBefore = "{\n" +
                    "  \"name\": \"Alice\",\n" +
                    "  \"age\": 30\n" +
                    "}";

После тройных кавычек
String jsonAfter = """
{
  "name": "Alice",
  "age": 30
}""";
